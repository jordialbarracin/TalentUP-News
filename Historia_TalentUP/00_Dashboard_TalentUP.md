# 🏠 Dashboard de TalentUP

Bienvenido a la "Caja Negra" de **TalentUP**. Esta bóveda de Obsidian contiene todo el registro de diseño, arquitectura y decisiones técnicas que tomamos para construir este agregador inteligente de noticias de Recursos Humanos desde cero.

## 🗂️ Índice del Proyecto

- [[01_Vision_y_Objetivos]]: Por qué creamos TalentUP y qué problema resuelve.
- [[02_Arquitectura_y_Descartes]]: Las herramientas No-Code que descartamos y por qué pivotamos a código propio.
- [[03_Diseño_UI_UX]]: La evolución visual, desde el "Dark Mode Bento" hasta el "Glassmorphism Google I/O".
- [[04_Motor_de_Rastreo_Python]]: Las entrañas técnicas del algoritmo de búsqueda.
- [[05_Roadmap_y_Futuro]]: Hacia dónde podemos llevar este proyecto (IA real, Notificaciones...).

## 📊 Ficha Técnica
- **Stack**: HTML5, CSS3, Vanilla JavaScript, Python 3.10.
- **Librerías Visuales**: Tailwind CSS (via CDN).
- **Despliegue**: GitHub Pages (Frontend) + GitHub Actions (Backend CI/CD).
- **Coste de Mantenimiento**: 0€ / mes.
- **Actualización**: Automática (cada 6 horas en la nube).
