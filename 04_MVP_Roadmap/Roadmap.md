# MVP Roadmap y Fases de Ejecución

## Fase 1 (MVP y Fontanería)
- [x] Definición de la estructura de conocimiento en Obsidian.
- [x] Creación de las cuentas de las herramientas a utilizar.
- [x] Configuración de la base de datos en Airtable (Tablas: Noticias, Fuentes).
- [ ] Conexión del primer flujo automatizado en Make.com con una o dos fuentes de prueba.

## Fase 2 (Front-end y Diseño)
- [ ] Conexión de la base de datos a Softr/Glide.
- [ ] Diseñar la interfaz visual pública (portada, categorías y tarjetas de noticias).

## Fase 3 (Enriquecimiento con IA)
- [ ] Integración opcional de la API de OpenAI en Make.com.
- [ ] Automatizar categorización de noticias.
- [ ] Mejorar los resúmenes generados mediante IA antes de publicarlos.

## Fase 4 (Lanzamiento y App)
- [ ] Adquisición de un dominio propio.
- [ ] Publicación oficial de la herramienta.
- [ ] Iteración hacia formato de aplicación móvil instalable.
